Boundary

Full profiling runs through the Python/DuckDB backend: local 127.0.0.1 by default, or an explicit trusted self-host bind. Hosted previews do not run profiler jobs.

Guided data-quality workflow

Choose whether to profile your CSV+DBML data or evaluate the tool on built-in faulty datasets.

Profile runs use the Python/DuckDB runner started from this repo. Evaluation runs curated local faulty datasets only and compares VSF output with seeded ground truth and the Great Expectations baseline state.
